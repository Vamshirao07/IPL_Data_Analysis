# IPL Analysis Dashboard (2008–2025)
## Project Overview

This project presents an interactive IPL (Indian Premier League) Analysis Dashboard developed using Microsoft Power BI. The dashboard provides comprehensive insights into IPL seasons, teams, player performances, and tournament statistics from 2008 to 2025.

The objective of this project is to transform raw cricket data into meaningful insights through data visualization and business intelligence techniques.

## Objectives

Analyze IPL season performance across teams and players

Track tournament statistics such as total matches, boundaries, and venues

Identify top performers including Orange Cap and Purple Cap holders

Display season winners and runner-up teams

Provide an interactive points table analysis

## Conclusion

The IPL Analysis Dashboard (2008–2025) demonstrates how sports data can be transformed into meaningful insights using Power BI. By applying data cleaning, modeling, and visualization techniques, the dashboard provides a comprehensive view of IPL tournament statistics, team performance, and player achievements.


Through interactive visualizations and key performance indicators, users can easily explore season-wise results, identify top-performing players, and analyze team standings. This project highlights the importance of data analytics in sports performance analysis and showcases practical skills in Power BI, DAX, and data storytelling.


Overall, the dashboard enables users to understand IPL trends, evaluate player contributions, and gain valuable insights into tournament performance across multiple seasons.
