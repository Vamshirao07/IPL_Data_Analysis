🏏 IPL Analysis Dashboard (2008–2025)
📊 Project Overview
This project presents an interactive IPL (Indian Premier League) Analysis Dashboard developed using Microsoft Power BI. The dashboard provides comprehensive insights into IPL seasons, teams, player performances, and tournament statistics from 2008 to 2025.
The objective of this project is to transform raw cricket data into meaningful insights through data visualization and business intelligence techniques.
🎯 Objectives
Analyze IPL season performance across teams and players
Track tournament statistics such as total matches, boundaries, and venues
Identify top performers including Orange Cap and Purple Cap holders
Display season winners and runner-up teams
Provide an interactive points table analysis

📈 Key Insights
Gujarat Titans players dominated both Orange Cap and Purple Cap statistics in the selected season.
Royal Challengers Bangalore emerged as IPL Champions.
Punjab Kings finished as Runner-Up.
The season recorded 1296 sixes and 2251 fours, indicating aggressive batting trends.
